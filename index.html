Formato del torneo

Este torneo usa sistema suizo: todos los jugadores disputan todas las rondas programadas y la clasificación se actualiza tras cada jornada.

Sorteo inicial

  • La ronda 1 se genera por sorteo aleatorio.
  • Los jugadores se mezclan automáticamente.
  • Se crean emparejamientos de 2 en 2.
  • Para evitar problemas, el torneo requiere número par de jugadores.

Rondas siguientes

  • Se emparejan jugadores con puntuación parecida.
  • El sistema intenta evitar rivales repetidos.
  • Si no hay combinación perfecta, usa la mejor alternativa disponible.
  • Cada nueva ronda solo se habilita cuando la actual está completa.

Puntuación

  • Victoria: 3 puntos
  • Empate: 1 punto
  • Derrota: 0 puntos

Desempates

  • Puntos
  • Diferencia de goles
  • Goles a favor
  • Menos goles encajados
  • Orden alfabético como último criterio visual

Consejo: tras la fase suiza puedes usar la pestaña “Fase final” para ver los clasificados provisionales y los cruces sugeridos.
